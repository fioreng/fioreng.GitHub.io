# CSS 基础语法

# 什么是 CSS？

CSS（Cascading Style Sheets，层叠样式表）是一种用于描述 HTML 或 XML 文档外观和格式的样式表语言。它控制网页的布局、颜色、字体、间距等视觉效果。

# CSS 语法

CSS 规则由选择器和声明块组成：

# 选择器

# 基础选择器

  • 元素选择器：选择指定类型的元素

  • 类选择器：选择带有指定类名的元素

  • ID 选择器：选择带有指定 ID 的元素

  • 通配符选择器：选择所有元素

# 复合选择器

  • 后代选择器：选择元素内的所有指定后代元素

  • 子选择器：选择元素的直接子元素

  • 相邻兄弟选择器：选择元素的下一个相邻兄弟元素

  • 通用兄弟选择器：选择元素的所有兄弟元素

# 常用属性

# 文本属性

  • color：设置文本颜色
  • font-size：设置字体大小
  • font-family：设置字体
  • font-weight：设置字体粗细
  • text-align：设置文本对齐方式
  • text-decoration：设置文本装饰

# 盒模型属性

  • width：设置元素宽度
  • height：设置元素高度
  • margin：设置外边距
  • padding：设置内边距
  • border：设置边框

# 背景属性

  • background-color：设置背景颜色
  • background-image：设置背景图片
  • background-size：设置背景图片大小
  • background-position：设置背景图片位置

# 布局属性

  • display：设置元素显示方式
  • position：设置元素定位方式
  • float：设置元素浮动
  • flex：设置弹性布局
  • grid：设置网格布局

# CSS 盒模型

CSS 盒模型由内到外依次是：

  1. 内容区域（content）：显示文本和图像的区域
  2. 内边距（padding）：内容与边框之间的空间
  3. 边框（border）：围绕内容和内边距的边界
  4. 外边距（margin）：元素与其他元素之间的空间

# CSS 优先级

CSS 优先级从高到低依次是：

  1. 内联样式：style 属性
  2. ID 选择器：#id
  3. 类选择器、属性选择器、伪类选择器：.class、[attr]、:hover
  4. 元素选择器、伪元素选择器：div、::before
